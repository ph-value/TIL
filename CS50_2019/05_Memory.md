# 메모리(Memory)

## 메모리 주소

### 16진법(Hexadecimal)

- 10진법, 2진법과 같이 16진법도 컴퓨터 과학에서 많이 쓰인다.
- 4bit씩 16진수로 변환후 `0x`를 붙여 16진수임을 알려준다.

| 진법 | 값 |
|---|---|
| 10진수  |  255 |
| 2진수  |  11111111 |
| 16진수 | 0xFF  |

### 메모리 주소

- 메모리상 주소를 받기 위해 `&` 연산자를 사용한다.

'''c
#include <stdio.h>
int main(void){
    int n = 50;
    printf("%i\n", &n);    // 0x7fgelsjf002a 와 같은 값이 출력된다.
}
'''

- 메모리 주소에 있는 값을 얻으려면 `*`연산자를 사용한다.
